# E-Commerce API's  for Node.js Application

This is a Node.js application with Swagger documentation to showcase API's.

## Overview of the Application
-   The application is built on Node.js, a popular and scalable platform for web development.
-   The application uses Swagger documentation, which provides an interactive UI for API understanding and testing.
-   The application supports Bearer token-based JWT authorization, which is a secure way to protect API access.
-   The application is designed to help developers streamline their development process and efficiently leverage APIs.
- **CRUD** operations are performed on Products, Brands, Categories and Cart.

## Prerequisites

Before you begin, ensure that you have the following installed on your system:

- Node.js (https://nodejs.org/)
- npm (Node Package Manager) - Usually installed along with Node.js

## Getting Started

1. **Clone the Repository**: Start by cloning or downloading this repository from your preferred version control system or source.

2. **Install Dependencies**: Navigate to the project's root directory in the terminal and install the required dependencies by running the following command:

```bash
npm install 
```
```bash
npm run dev
```

### So, the application should listen to the port 8080 as mentioned in the application.

## Using  Swagger-UI
Navigate to the Swagger UI
```
http://localhost:8080/docs/
```
> **Note:** Before using the API's Authentication is Mandatory.
